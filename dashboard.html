💡 Dica para contornar limitações do Supabase

Por padrão, o Supabase limita o cadastro direto pelo front-end para evitar abusos (como limite de taxa de e-mails de confirmação e políticas de signup público). Caso receba algum erro ao cadastrar o operador aqui, você pode criá-lo em segundos diretamente pelo Painel do Supabase (Authentication → Users → Add User → Create User) marcando a opção "Auto Confirm". Depois, basta clicar em "Atualizar Lista" aqui no dashboard para configurar o nome, meta e função dele.
